# Handbook

Repositorio personal de notas, guías, scripts y recursos relacionados con seguridad informática y pentesting.

## Estructura
- `cheatsheets/` → referencias rápidas
- `scripts/` → scripts útiles
- `guides/` → guías paso a paso
- `security/` → prácticas de seguridad
- `notes/` → apuntes personales
- `resources/` → bibliografía y enlaces
