# 🔐 Extensiones Gratuitas de Seguridad y Privacidad para Navegadores

Estas extensiones son **gratuitas**, aptas para **uso comercial y empresarial**, y recomendadas para mejorar la seguridad y privacidad en la navegación web.

---

## 📌 Lista de Extensiones

- [**uBlock Origin**](https://github.com/gorhill/uBlock)  
  Bloqueador de anuncios y rastreadores.  
  
  - Compatibilidad: Firefox, Chrome, Edge, Opera.  
  - Licencia: **GPLv3 (código abierto, uso comercial permitido)**.

- [**Privacy Badger (EFF)**](https://privacybadger.org/)  
  Bloquea rastreadores invisibles de manera automática.  
  
  - Compatibilidad: Firefox, Chrome, Edge.  
  - Licencia: **GPLv3 (código abierto, uso comercial permitido)**.

- [**AdGuard AdBlocker**](https://adguard.com/en/adguard-browser-extension/overview.html)  
  Bloqueo avanzado de publicidad y rastreadores.  
  
  - Compatibilidad: Firefox, Chrome, Edge, Safari, Opera.  
  - Licencia: **Gratis para uso personal y comercial** (versión premium opcional).

- [**DuckDuckGo Privacy Essentials**](https://duckduckgo.com/privacy)  
  Bloquea rastreadores y fuerza cifrado HTTPS.  
  
  - Compatibilidad: Firefox, Chrome, Edge, Safari.  
  - Licencia: **Gratis, uso comercial permitido**.

- [**Ghostery**](https://www.ghostery.com/)  
  Bloquea rastreadores y muestra estadísticas de seguimiento.  
  
  - Compatibilidad: Firefox, Chrome, Edge, Opera, Safari.  
  - Licencia: **Gratis, uso comercial permitido** (Pro opcional).

- [**NoScript Security Suite**](https://noscript.net/)  
  Bloquea JavaScript, Java y Flash maliciosos.  
  
  - Compatibilidad: Solo Firefox.  
  - Licencia: **GPL (código abierto, uso comercial permitido)**.

- [**HTTPS Everywhere (EFF)**](https://www.eff.org/https-everywhere)  
  Fuerza conexiones seguras HTTPS.  
  
  - Compatibilidad: Firefox, Chrome, Opera.  
  - Nota: Ya integrado en la mayoría de navegadores modernos.  
  - Licencia: **GPL (código abierto, uso comercial permitido)**.

- [**Malwarebytes Browser Guard**](https://www.malwarebytes.com/browserguard)  
  Bloquea malware, phishing y sitios fraudulentos.  
  
  - Compatibilidad: Firefox, Chrome, Edge.  
  - Licencia: **Gratis, uso comercial permitido**.

- [**Bitdefender TrafficLight**](https://www.bitdefender.com/solutions/trafficlight.html)  
  Escaneo de enlaces y descargas contra malware.  
  
  - Compatibilidad: Firefox, Chrome, Safari.  
  - Licencia: **Gratis, uso comercial permitido**.

---

## ✅ Recomendaciones por Navegador

- **Firefox** → uBlock Origin + Privacy Badger + NoScript + Malwarebytes Browser Guard.  
- **Chrome/Edge** → uBlock Origin + Privacy Badger + DuckDuckGo Privacy Essentials + Malwarebytes Browser Guard.  
- **Safari** → Ghostery + DuckDuckGo Privacy Essentials + Bitdefender TrafficLight.  
- **Brave** → Ya incluye bloqueadores integrados, opcional añadir Privacy Badger o Malwarebytes Browser Guard.  

---

## ⚠️ Consideraciones

- Todas las extensiones listadas son **gratuitas y aptas para uso comercial/empresarial**.  
- Algunas ofrecen versiones premium
