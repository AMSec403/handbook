# 🚀 DNS para Android

Listado de DNS seguros y gratuitos que bloquean anuncios, rastreadores y ofrecen perfiles de seguridad.  
Todos soportan **DoH/DoT** y se configuran en Android desde **Configuración → Red e Internet → DNS privado**.

---

## 🔒 ControlD FreeDNS

[Free and Public DNS Servers](https://controld.com/free-dns#quick-setups)

- `ads.control-dns.com` → Bloquea anuncios y rastreadores.
- `social.control-dns.com` → Bloquea redes sociales.
- `family.control-dns.com` → Bloquea malware, anuncios, rastreadores y contenido adulto.

---

## 🔒 Mullvad DNS

[DNS over HTTPS and DNS over TLS](https://mullvad.net/en/help/dns-over-https-and-dns-over-tls)

- `adblock.dns.mullvad.net` → Bloquea anuncios y rastreadores.
- `base.dns.mullvad.net` → Bloqueo básico de malware.
- `extended.dns.mullvad.net` → Bloqueo extendido de rastreadores y publicidad.
- `family.dns.mullvad.net` → Bloquea anuncios + contenido adulto.
- `all.dns.mullvad.net` → Bloquea malware, anuncios, rastreadores y contenido adulto.
